How to contribute
=================

## Prerequisites

* PHP >= 5.4
* [composer](https://getcomposer.org/)

### Install the dependencies

```sh
$ composer install
```

## Launch it

```sh
$ php -S localhost:8000 install_dev.php
```

And now just call [`http://localhost:8000`](http://localhost:8000).

## Build the installer

```sh
$ bin/console installer:build
```
